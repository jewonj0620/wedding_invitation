# Wedding Invitation

Mobile wedding invitation for Jiwon Choi and Jewon Jeon.

## Local Development

```bash
npm install
npm run dev
```

## Checks

```bash
npm run lint
npm run build
```

## GitHub Pages

This project is configured for a GitHub repository named `wedding_invitation`.

Expected public URL:

```text
https://<github-username>.github.io/wedding_invitation/
```

Deployment is handled by `.github/workflows/deploy.yml`. In GitHub, set the
repository Pages source to **GitHub Actions**.
